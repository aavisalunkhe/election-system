About the Project

Project Description

This Election Management System is a comprehensive database application built with Flask and SQLite that models the Indian election system with full CRUD operations, interactive data management, and real-time analytics.

Key Features

  • CRUD Operations: Create, Read, Update, Delete functionality for all entities
  • Interactive Voting: Simulate vote casting with real-time updates
  • Results Management: Add and manage election results dynamically
  • Data Export: Export data to JSON format for backup
  • Custom Queries: Execute pre-defined SQL queries
  • Real-time Analytics: Live charts and statistics
  • Search & Filter: Advanced search across all tables

Technology Stack

  • Backend: Flask (Python)
  • Database: SQLite3
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Visualization: Chart.js
